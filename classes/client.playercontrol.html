reportDeadBody

  • reportDeadBody(body: number | CustomNetworkTransform | PlayerControl | PlayerPhysics | PlayerData | "emergency"): void
    • Defined in core/js/lib/component/PlayerControl.d.ts:349

    Ask the host to start a meeting either for a reported body or for an emergency.

    see

    {PlayerControl.startMeeting} If the player is the host and this can happen immediately.

    example
    // Start an emergency meeting.
if (client.amhost) {
  client.me.control.startMeeting("emergency");
} else {
  client.me.control.reportDeadBody("emergency");
}

    // Start an meeting reporting another player.
if (client.amhost) {
  client.me.control.startMeeting(player);
} else {
  client.me.control.reportDeadBody(player);
}

    Parameters

    • body: number | CustomNetworkTransform | PlayerControl | PlayerPhysics | PlayerData | "emergency"

      Either the body to report or whether it was an emergency.

    Returns void
