findGames

  • findGames(maps?: number | MapID[], impostors?: number, language?: LanguageID): Promise<GameListing[]>
    • Defined in client/ts/lib/client.ts:903

    Search for public games.

    example
    // Search for games and join a random one.
const client = new SkeldjsClient("2021.3.5.0");

await client.connect("EU", "weakeyes");

const games = await client.findGames();
const game = games[Math.floor(Math.random() * games.length)];

const code = await game.join();

    Parameters

    • Default value maps: number | MapID[] = 7

      The maps of games to look for. If a number, it will be a bitfield of the maps, else, it will be an array of the maps.

    • Default value impostors: number = 0

      The number of impostors to look for. 0 for any amount.

    • Default value language: LanguageID = LanguageID.All

      The language of the game to look for, LanguageID.All for any.

    Returns Promise<GameListing[]>

    An array of game listings with the data and a GameListing.join method to join the game.
