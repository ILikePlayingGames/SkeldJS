Class PlayerControl<RoomType>

setImpostors

  • setImpostors(impostors: PlayerData<Hostable<any>>[]): Promise<void>
    • Defined in core/lib/objects/PlayerControl.ts:425

    Set the impostors of the room. Usually called straight after a game is started via the InnerShipStatus.selectImpostors method. This is a host operation on official servers.

    Will remove any existing impostors unless they are included in the array.

    Emits a player.setimpostors event.

    example
    // Set everyone with "Judas" in their name as the impostor.
awiat client.me.control.setImpostors(
  [...this.room.players.values()]
    .filter(player => player.info.name.includes("Judas"))
);

    Parameters

    • impostors: PlayerData<Hostable<any>>[]

      An array of impostors to set.

    Returns Promise<void>
